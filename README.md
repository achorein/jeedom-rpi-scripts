# Jeedom-scripts

Scripts Python pour Raspberry Pi

# Matériel

- Mesure de la température : capteur TMP36
- Mesure de luminosité : photo-resistance
- Convertisseur analogique vers numérique : MCP3008-i/p DIP-16
